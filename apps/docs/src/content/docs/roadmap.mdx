This document outlines the planned trajectory for Monque, moving from a robust job queue to a comprehensive orchestration platform.

## Core Package (`@monque/core`)

<Roadmap items={[
  {
    title: 'The "Management" Release',
    version: 'v1.1.0',

    description: 'Foundational APIs needed for external tooling like dashboards.',

      'Statistics & Aggregation APIs'
    ]
  },
  {
    title: 'The "Orchestration" Release',
    version: 'v1.2.0',
    status: 'planned',
    description: 'Enable code-first orchestration for complex, multi-step processes.',
    features: [
      'Workflows Engine with runJob() helper',
      'State Management (pause, resume, cancel workflows)',
      'Expanded Event Bus for full lifecycle observability'
    ]
  },
  {
    title: 'The "Pipeline" Release',
    version: 'v1.3.0',
    status: 'planned',
    description: 'Support user-defined flows via configuration rather than code.',
    features: [
      'Declarative JSON-based pipelines',
      'Control Flow Primitives (if, switch, parallel, map, wait)',
      'Pipeline execution events'
    ]
  },
  {
    title: 'The "Enterprise" Release',
    version: 'v1.4.0',
    status: 'planned',
    description: 'Features to support SaaS platforms and high-scale deployments.',
    features: [
      'Multi-Tenancy / Namespacing',
      'Global & Per-queue Rate Limiting',
      'Priority Queues'
    ]
  }
]} />
