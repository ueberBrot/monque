# API Contracts: @monque/tsed

This directory contains TypeScript interface definitions that form the public API contract for the `@monque/tsed` package.

## Files

- `decorators.ts` - Decorator function signatures and options
- `config.ts` - Configuration interface for TsED augmentation
- `services.ts` - MonqueService public API
- `metadata.ts` - Internal metadata structures

## Usage

These contracts serve as the source of truth for:
1. Implementation in `packages/tsed/src/`
2. Documentation generation
3. Consumer type safety
